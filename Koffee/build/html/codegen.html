Code generation functions¶

The file codegen.py includes the code generation functions which takes in the abstract syntax trees and generates the corresponding java code.

codegen.codegen(input_ast, tabs=0, wnum=0)¶

The main code generation function. It takes as input an AST, and returns its corresponding java code. It is called recursively on the branches of the syntax tree.

Args:
input_ast (ast): if its not an AST, then return nothing, else generate code. tabs (int): indentation for generated java program.
Returns:
generated_code (str): java code as a string.
codegen.get_class_decl(app_name)¶

Function to generate the Java class declaration.

codegen.get_class_method(app_name)¶

class generation method with gvh input

codegen.get_package_decl(app_name)¶

Function to generate package declaration from application name.

function for wrapping the event code in the loop.
